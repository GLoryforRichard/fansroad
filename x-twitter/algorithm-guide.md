# 📱 X (Twitter) 内容风格与算法偏好

## 🎯 算法核心要点 (2025)

### 流量分发机制
- **回复率权重高**：获得 3 条回复 > 发出 1 条回复
- **实时性重要**：前 30 分钟的数据决定初始流量
- **社区权重**：在 X Communities 发帖是涨粉最快方式

### 爆款内容特点

| 类型 | 说明 | 例子 |
|------|------|------|
| 🗣️ 提问 | 引发讨论的问题 | "你们觉得 XXX 怎么样？" |
| 💡 观点 | 鲜明的个人见解 | "XXX 是错的，原因是..." |
| 🎭 吐槽 | 幽默的抱怨 | "为什么 XXX 总是..." |
| 📊 数据 | 有理有据的分析 | 附图表/数据的研究 |
| 🎬 切片 | 有趣的内容片段 | 视频/文章的关键点 |
| 🔥 热点 | 紧跟时事 | 热门话题 + 独特角度 |

### 最佳实践
1. **短平快** - 文字精炼，观点鲜明
2. **开头抓人** - 前 20 字就要吸引注意力
3. **引发互动** - 提问、投票、争议性观点
4. **使用话题标签** - 但不要过多（1-3个）
5. **图片/链接** - 带媒体的推文曝光更高
6. **一致性** - 固定时间发帖，建立期待

### 发布时间
- 早 7-9 点（通勤）
- 午 12-13 点（午休）
- 晚 18-21 点（下班后）

```
[开头抓人句子 - 10-20字]
[核心内容 - 50-100字]
[引发互动 - 提问/投票]
#标签1 #标签2
```
